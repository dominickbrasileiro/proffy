## :information_source: How To Use

```bash
# Go into the web folder (from the root of the repository)
$ cd web

# Rename .env.example to .env
$ mv .env.example .env

# Set the environment variables in .env

# Install dependencies
$ yarn install

# Run the web app
$ yarn start

# React App is running on port 3000.
```
